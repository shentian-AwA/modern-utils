# Modern Utils 🛠️

A collection of modern, high-performance TypeScript utility functions.

## Features
- 🚀 **Lightweight**: Zero dependencies.
- 🛡️ **Type Safe**: Written in TypeScript.
- 📦 **Tree Shakable**: Use only what you need.

## Installation
```bash
npm install @shentian-AwA/modern-utils
```

## Usage
```typescript
import { debounce } from '@shentian-AwA/modern-utils';

const handleSearch = debounce((query: string) => {
  console.log('Searching for:', query);
}, 500);
```
